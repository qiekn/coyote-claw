## 文档

目前使用了 [zensical]() 生成文档并部署到了 Github Pages → https://qiekn.github.io/coyote-claw/

如果想要本地查看文档，首先使用安装 python 的包管理器 [uv](https://docs.astral.sh/uv/)，然后可以使用下面命令在本地打开文档。更多详情可以看 [zensical 官网文档](https://zensical.org)

```bash
cd docs
uv init
uv add --dev zensical
uv run zensical serve --open
```
